How to Start (Quick Setup)

Goal

Set up the localization tool quickly with one-click component setup and add your first languages.

Prerequisites

  • AnKo Localization Tool installed
  • Unity project opened

Steps

  1. Install the tool via Package Manager or manual import
  2. Open the Localization Tool Window (Tools > AnKo Localization Tool)
  3. Click "One-Click Component Setup"
  4. Add your first two languages in the Settings tab

Demo

How to Localize Text (UGUI / TMP)

Goal

Add localized text to UI elements using UGUI or TextMeshPro components.

Prerequisites

  • Localization tool installed and configured
  • UI element (Button, Text, etc.) in your scene

Steps

  1. Select a UI element with text in your scene
  2. Add the "LocalizedText" component
  3. Create a new key or select an existing one
  4. Enter translations for each language

Demo

How to Find Keys (Parsing)

Goal

Automatically find and register text elements in your scenes for localization.

Prerequisites

  • Scenes with text elements set up
  • Localization tool window open

Steps

  1. Navigate to the Content tab in the Localization Tool
  2. Select scenes to parse
  3. Click the "Scan" button
  4. Review detected keys in the table

Demo

How to Translate (Manual & Auto)

Goal

Translate your content manually or automatically using machine translation services.

Prerequisites

  • Keys identified through parsing
  • (For auto-translation) API keys for Google Translate or DeepL

Steps

  1. Open the Translation Editor
  2. Enter translations manually in the table
  3. Or select text and use Machine Translation
  4. Review and refine translations

Demo

How to Preview in Editor

Goal

Preview your localized content in the editor without running the game.

Prerequisites

  • Translations entered for at least two languages
  • Localization tool configured

Steps

  1. Open the Live Preview panel
  2. Select a language from the dropdown
  3. Observe text changes in the Game view
  4. Switch between languages to verify translations

How to Setup AI Translation (OpenAI/Gemini)

Goal

Configure AI-powered translation using OpenAI or Gemini services.

Prerequisites

  • API key from OpenAI or Google Cloud
  • Localization tool installed

Steps

  1. Create a new Translation Profile in Settings
  2. Enter your API key
  3. Select service provider (OpenAI/Gemini)
  4. Translate text using AI in the editor

Demo

How to Use AI Assistant

Goal

Improve your translations using AI-powered assistance features.

Prerequisites

  • AI translation profile configured
  • Text to be improved in the translation table

Steps

  1. Select text in the translation editor
  2. Right-click to open context menu
  3. Choose AI options: "Fix Grammar", "Shorten for UI", "Make Formal"
  4. Review and accept suggested changes
